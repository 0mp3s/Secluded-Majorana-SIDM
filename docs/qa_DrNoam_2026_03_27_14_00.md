## 2. Methodology Context

The project computes the **velocity-dependent self-interaction cross section** of a Majorana DM fermion χ exchanging a real scalar mediator φ via

$$\mathcal{L}_{\rm int} = -\frac{y}{2}\bar\chi\chi\phi, \quad \alpha = \frac{y^2}{4\pi}, \quad V(r) = -\frac{\alpha\, e^{-m_\phi r}}{r}$$

Phase shifts are computed via the **Variable Phase Method (VPM)** — a full partial-wave integration — to obtain the Majorana-symmetrised transfer cross section:

$$\frac{\sigma_T}{m} = \frac{2\pi}{k^2 m_\chi}\sum_\ell w_\ell (2\ell+1)\sin^2\delta_\ell, \quad w_\ell = \begin{cases} 1 & \ell \text{ even} \\ 3 & \ell \text{ odd} \end{cases}$$

The relic density uses a full Boltzmann integrator with s-wave annihilation ⟨σv⟩ = πα²/4m²_χ.

### 3.4 ⚠️ HIGH — `cp_channel_tension.py` Uses `4π/k²` Prefactor for Internal Channels
**Location:** `cross_checks/cp_channel_tension.py`, line 110  
```python
prefactor = 4.0 * math.pi / (k * k) * GEV2_TO_CM2 / (m_chi * GEV_IN_G)
```
This function returns **(σ_even, σ_odd)** partial sums with the **Dirac prefactor** (4π), not the Majorana prefactor (2π). When the caller reconstructs σ_T^Maj as `sigma_even + 3 * sigma_odd`, the result is:

$$\sigma_T^{\rm Maj} = \frac{4\pi}{k^2}\left(\Sigma_{\rm even} + 3\Sigma_{\rm odd}\right) \neq \frac{2\pi}{k^2}\left(\Sigma_{\rm even} + 3\Sigma_{\rm odd}\right)$$

This is **a factor of 2 error** in the CP-decomposition absolute values. The **ratio** σ_even / σ_odd (which is what the script primarily plots) is unaffected, but the reconstructed total σ_T would be double the correct value.

**Scope:** This only affects `cp_channel_tension.py`. The primary `sigma_T_vpm` in `v22_raw_scan.py` is correct. All scan, relic, χ², and MCMC results are unaffected.

**Action required:** Change line 110 to:
```python
prefactor = 2.0 * math.pi / (k * k) * GEV2_TO_CM2 / (m_chi * GEV_IN_G)
```
Or, document clearly that the function returns "half-channels normalised to Dirac convention" and all reconstructed totals must be halved.

### 3.5 ✅ Boltzmann Relic — s-wave Cross Section Formula (v27_boltzmann_relic.py, L75–84)

### 4.7 ⚠️ MED — MCMC One-Sided Limits Treatment
**Location:** `stats_mcmc/opusB_run_mcmc.py`, `_log_prob_worker` function  
The MCMC log-probability uses **symmetric Gaussian likelihood** for all 13 observations, including the Bullet Cluster (one-sided, v=4700 km/s) and Harvey+15 (one-sided, v=1000 km/s):

The χ² fit in `observations/chi2_fit.py` correctly skips the penalty for one-sided limits when theory < hi. The MCMC likelihood does **not** replicate this — it applies a symmetric penalty even when theory is safely below the limit. This is a systematic inconsistency between the χ² fit and the MCMC, which could bias the posterior away from high-velocity (cluster) parameter space that is legitimately allowed.

**Severity:** Medium. The one-sided limits are very weak (Bullet < 1.25, Harvey < 0.47) and the model predictions at these velocities are well inside them, so the bias is likely small in practice. However, it violates formal consistency between the two analysis tracks.

**Action required:** In `_log_prob_worker`, add the same one-sided handling:
```python
if lo == 0.0 and theory <= hi:
    continue  # no penalty for satisfied one-sided upper limit
```

## 8. Theory Consistency Review

### 8.1 The Lagrangian and the Potential

The Lagrangian $\mathcal{L}_{\rm int} = -\frac{y}{2}\bar\chi\chi\phi$ for a Majorana fermion gives a scalar (not pseudoscalar) exchange. In the non-relativistic limit, this yields an **attractive** Yukawa potential for both spin-singlet **and** spin-triplet scattering — which is correct for scalar exchange (all signs align attractively). The factor 1/2 in the coupling is the Majorana convention to avoid double-counting.

**The mixed-coupling issue:** Peer review flagged that pure-scalar Yukawa to Majorana fermions gives **p-wave suppressed** annihilation χχ → φφ at threshold. This is addressed in `methodology.md §9.2` and `mixed_coupling/MixedMajorana.md`. The resolution (mixed scalar + pseudoscalar coupling) restores s-wave annihilation while the VPM scattering sector (which depends only on the potential, not on the coupling structure) is unchanged. The annihilation formula ⟨σv⟩ = πα²/4m²_χ is reinterpreted as πα_sα_p/4m²_χ with α_s ≈ α_p = α.

**My assessment:** The mixed-coupling resolution is valid but introduces an implicit assumption that α_s ≈ α_p (equal scalar and pseudoscalar couplings). This is not a necessary consequence of the model — it's a parameter choice that happens to be consistent with current data. The preprint should be clearer that α = √(α_sα_p) is a geometric mean of two independent parameters, and the constraint α_s = α_p is an ansatz (motivated by simplicity) rather than a derived result.

### 8.2 The s-wave Claim and the Pauli Principle

The preprint claims (§3.2): "The factor of 3 for odd partial waves arises from spin statistics: ... triplet spin state (weight 3) combines with odd orbital angular momentum to form the antisymmetric total wave function."

This is correct but requires care. For identical Majorana fermions (spin-1/2), the total wave function must be antisymmetric:
- Singlet spin (S=0, symmetric under particle exchange) × even-ℓ orbital (antisymmetric under r→-r, which for ℓ even gives +1 under exchange) = **antisymmetric** ✅
- Wait — this needs careful sign tracking:

Under particle exchange (1↔2), the spatial wave function transforms as (-1)^ℓ, and the spin state as (-1)^(S+1). For total antisymmetry: (-1)^ℓ × (-1)^(S+1) = -1 → (-1)^(ℓ+S) = 1 → ℓ + S = even.

So: (even ℓ, S=0), (even ℓ, S=2)... but S ≤ 1 for two spin-1/2 particles. Valid combinations: **(even ℓ, S=0 singlet)** and **(odd ℓ, S=1 triplet)**. Weights: singlet has 1 state, triplet has 3 states. ✅ The code correctly applies w=1 for even ℓ and w=3 for odd ℓ.

### 8.3 The Identical-Particle σ_T = σ_el Theorem

The analytic proof in `cross_checks/transfer_vs_elastic.py` is valid. The key insight — that within each spin channel, all contributing partial waves share the same parity, so the differential cross section is symmetric under θ ↔ π−θ, making the ⟨cosθ⟩ integral vanish — is a genuine and non-trivial result. This is numerically confirmed at the 10⁻¹² level.

### 8.4 Relic Density Consistency

The "island of viability" at m_χ ∈ [10, 100] GeV, m_φ ∈ [7.6, 14.8] MeV arises from the intersection of:
1. σ/m(30) ≥ 0.5 cm²/g (dSph lower bound) — requires λ > ~0.5
2. σ/m(1000) < 0.47 cm²/g (cluster upper bound) — requires λ not too large
3. Ωh² = 0.120 — fixes α given (m_χ, m_φ)

The relic constraint scales as α ∝ m_χ (from ⟨σv⟩ ∝ α²/m²_χ = const). Combined with the SIDM condition (λ = αm_χ/m_φ ∈ reasonable range), the viable island is a real consequence of parameter interplay. I see no logical inconsistency.

### 8.5 The Secluded Sector and ΔN_eff

The claim that a real scalar φ with m_φ ~ 10 MeV contributes ΔN_eff ≈ 0 requires verification. The argument: φ is non-relativistic at BBN (T_BBN ~ 1 MeV << m_φ), so it is Boltzmann-suppressed in the dark plasma. With no portal to the SM, φ never thermalises with the SM sector, and its phase-space density at CMB is negligible. This is correct for the secluded sector. ✅

For the Higgs-portal scenario (sin θ > 0), there would be constraints, but the secluded sector genuinely avoids this. ✅

## 10. Missed Opportunities / Recommendations

1. **CMB direct detection constraint on s-wave annihilation:** The secluded sector model has χχ → φφ with φ remaining dark. However, if φ eventually decays slowly (via gravity or higher-order loops), the energy injection could be constrained by Planck CMB spectral distortions. A calculation of the effective ⟨σv⟩ × B_EM bound (even conservatively) would strengthen the paper.

2. **Perturbativity bound α < 4π (unitarity self-consistency):** The maps shows α values up to ~0.05. A brief note that perturbativity requires α ≪ 1 (satisfied throughout) and the NDA coupling bound α < 4π is trivially satisfied would close a potential reviewer objection.

3. **λ > 50 MCMC exclusion region:** As noted in §7.3, the MAP point has λ = 48.6. Raising the hard cutoff to 80 would allow the MCMC to explore the MAP neighbourhood without artificial truncation. This is especially important for MAP_relic consistency.

4. **Signal comparison with sidmkit:** The preprint mentions that sidmkit (the Hulthén approximation pipeline) fails at λ ≳ 1. A brief numerical comparison table (VPM vs sidmkit at the 17 BPs) would make this claim quantitative and citation-worthy.

5. **Gravothermal collapse timescale (`predictions/gravothermal/`):** The prediction t_grav/t_age for 8 dSphs is mentioned in V9 results but not yet updated for V10 BPs. This is a falsifiable prediction that could distinguish the model from simpler SIDM models with constant σ/m.

6. **Bootstrap or jackknife error on χ²:** The current χ² = 1.22 at best-fit uses published observational error bars which are themselves uncertain. A sensitivity study (perturbing each error bar by ±30%) would quantify how sensitive the χ²/dof is to meta-uncertainties in the dataset.

## 11. Personal Opinion on the Model

I will state this clearly: **this is a physically well-motivated, economical model with genuine predictive power.**

The three key virtues are:

**1. Economy:** Three parameters (m_χ, m_φ, α) explain phenomena spanning four decades in velocity. The secluded-sector structure is not an ad-hoc choice but a forced conclusion from the direct-detection bound calculation. Models that need 6+ parameters to fit the same data should be viewed skeptically by comparison.

**2. The island is a genuine prediction:** The intersection of cosmological relic density and astrophysical SIDM viability is not trivially guaranteed — there is no reason a priori why the coupling value needed for Ωh² = 0.120 should fall inside the SIDM window. That it does (and in a well-defined, extended island) is a non-trivial coincidence that the model explains as the "sweet spot" of the Yukawa resonance structure.

**3. The Majorana vs. Dirac fingerprint:** The non-monotonic ratio R(v) = σ_Maj/σ_Dir with R > 1 at intermediate velocities (v ~ 40–95 km/s) and R = 4 at low velocities is a genuine quantum-statistical prediction. The proof that σ_T = σ_el exactly for identical Majorana fermions is a clean analytic result that I verified independently.

**Where I am less convinced:**

- The mixed scalar+pseudoscalar coupling ansatz (α_s = α_p = α) lacks a theoretical justification. This is the weakest point of the V10 model. If α_s ≪ α_p or vice versa, the scattering and annihilation sectors decouple, the island could shift or disappear, and the VPM scattering computation would need corrections from the pseudoscalar potential (which is velocity-dependent and more complex).

- The claim χ²/dof ≈ 0.12 for the **free** fit and χ²/dof ≈ 0.13 for the **relic-constrained** fit are suspiciously close. For a model with 3 vs 2 free parameters fitting 13 data points, one expects the free fit to outperform the relic-constrained one by a statistically meaningful margin. The near-equality suggests either (a) the data strongly prefer the relic-constrained parameter space, or (b) the optimiser in chi2_fit.py didn't fully explore the unconstrained space. I lean toward (a) being physically correct — it would be a strong result — but it deserves explicit discussion in the paper.

- BBN and φ stability: with sin θ = 0 exactly, φ never decays and accumulates in the universe. If φ has any tiny residual coupling (even gravitationally induced), there could be lifetime constraints. The paper correctly notes that for a strictly secluded sector this is a prediction (null signal), but a brief discussion of the φ relic density and its possible gravitational effects on structure formation would strengthen the cosmological sector.
